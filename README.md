# AWS Resume Capstone Project
A personal resume website deployed using AWS services including S3, CloudFront, Route 53, and ACM.
Live Site: https://josevaldezfernandez.com/

## Tech Stack
- HTML, CSS, JavaScript
- AWS S3 (static hosting)
- AWS CloudFront (CDN)
- AWS Route 53 (DNS)
- AWS Certificate Manager (HTTPS)

## Architecture
User → Route 53 → CloudFront → S3

## Features
- Static website hosting using S3
- Custom domain with Route 53
- HTTPS enabled via ACM
- Global content delivery using CloudFront

## What I Learned
- How to deploy static websites using AWS S3
- How DNS works with Route 53
- How to configure CloudFront for performance and HTTPS
- Basics of cloud architecture and CDN usage
